# Frozen production baseline

This result package freezes the current production source/config boundary for repository construction:

- Semantic protocol: `readable_grounded_core_v1u`
- Renderer: `adaptive_v1e`
- Handler: `EMAIL_WATCHDOG_OUTBOX_NONBLOCKING_BACKOFF_V1`
- Mobile visual E2E: accepted from the phone screenshot in the current chat
- Email Watchdog business outbox: nonblocking, persistent and bounded-backoff
- Weixin adapter and WeChat Enhance: external ownership boundary; excluded from the Email Watchdog source seed

The freeze is a source and configuration baseline. It is not a GitHub release, final distributable acceptance, or approval to modify production.
